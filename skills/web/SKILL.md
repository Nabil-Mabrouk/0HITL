# WEB SKILL
Cette competence te permet de rechercher des informations sur le web et d'extraire du contenu HTTP sans passer par du bash artisanal.

# DIRECTIVES
1. Utilise `search_web` pour trouver rapidement des pages pertinentes avant de fetch une URL precise.
2. Utilise `fetch_url` pour recuperer un apercu structure d'une page ou d'une ressource HTTP.
3. Utilise `extract_page_text` quand tu veux surtout le texte lisible d'une page HTML.
4. Utilise `extract_links` pour explorer la structure d'une page ou reperer les liens utiles.
5. Garde les usages en lecture seule ; cette skill ne doit pas etre utilisee pour poster des donnees sensibles vers des sites externes.
6. Si l'utilisateur demande une verification en ligne ou une information recente, active cette skill tres tot dans la mission.
